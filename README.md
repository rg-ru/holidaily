# holidaily

Statische Landingpage fuer holidaily pools.

## Struktur

- `index.html`: Seitenstruktur
- `styles.css`: Layout und Design
- `assets/`: Bilder und statische Assets
- `playwright-audit/`: Screenshots, Audits und Design-Vergleiche
- `playwright-audit/hero-iterations/`: Hero-Varianten und Verifikationsbilder

## Lokal starten

Die Seite kann direkt im Browser ueber `index.html` geoeffnet werden.

## Deployment

Das Projekt ist fuer statisches Hosting ueber GitHub Pages geeignet.
